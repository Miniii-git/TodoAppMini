# Android Todo App

## Overview

Android Todo App is a task management application developed using Apache Cordova. It allows users to create, manage, and organize daily tasks through a simple and intuitive interface. 

---

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Simple and intuitive user interface
- Mobile-friendly design
- Local storage for task persistence

---

## Technologies

- Apache Cordova
- JavaScript
- HTML
- CSS

---

## Deployment

The application was successfully built as an Android APK using Apache Cordova. <br>
Online Demo : https://todoappmini.netlify.app/
